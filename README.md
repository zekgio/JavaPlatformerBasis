# Simple Platformer (Java)

A demo of a simple platformer game, written in Java

![scrolling](https://github.com/user-attachments/assets/12e2d4e8-da44-414c-9d3f-bd489f68dcb1)

## Features
- **Java Standard**
- **Maven**

## Controls
- [A] and [D] to move
- [SPACE] to jump
- [M] to attack
- [N] for special attack

## Build & Run

### Prerequisites
- **Java JDK 25**

### Instructions (Terminal)
```bash
git clone https://github.com/zekgio/JavaPlatformerBasis.git
cd JavaPlatformerBasis
./mvnw compile exec:java (or mvnw compile exec:java on Windows)
```
